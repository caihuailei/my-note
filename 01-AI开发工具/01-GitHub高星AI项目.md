## 1️⃣ agency-agents

### 📋 基本信息
- **完整名称**: `msitarzewski/agency-agents`
- **GitHub 链接**: https://github.com/msitarzewski/agency-agents
- **Star 数量**: ⭐ **60,206**
- **中文版本**: `jnMetaCode/agency-agents-zh` (⭐ 2,088)

### 🎯 主要功能
**"一个完整的 AI 代理机构，触手可及"**

- 提供 180+ 个即插即用的 AI 专家人设
- 覆盖 17 个不同部门（前端、Reddit 社区、Whisper 注入、现实检测等）
- 每个 Agent 都是具有个性、流程和既定交付成果的专业专家
- 支持多种工具：Claude Code / OpenClaw / Cursor / Trae 等 11 种工具

### 💡 技术特点
- **模块化设计**: 每个 Agent 独立可插拔
- **多工具支持**: 兼容主流 AI 开发工具
- **中文本地化**: 提供完整的中文版本
- **部门覆盖广**: 从技术开发到社区运营全覆盖

### 🎨 适用场景
- 企业级 AI 代理部署
- 多角色 AI 协作系统
- 自动化业务流程
- 社区管理和内容创作

## 2️⃣ autoresearch

### 📋 基本信息
- **完整名称**: `karpathy/autoresearch`
- **GitHub 链接**: https://github.com/karpathy/autoresearch
- **Star 数量**: ⭐ **51,141**
- **作者**: Andrej Karpathy (前 Tesla AI 负责人)

### 🎯 主要功能
**"AI 代理在单 GPU 上进行纳米芯片训练自动研究"**

- 完全自主的研究循环
- 从想法到论文的自动化流程
- 单 GPU 即可运行纳米级训练
- 自动迭代和验证机制

### 💡 技术特点
- **自主研究**: AI 自动执行研究任务
- **轻量级**: 单 GPU 即可运行
- **自动迭代**: Modify → Verify → Keep/Discard → Repeat
- **高效**: 专为纳米级训练优化

### 🎨 适用场景
- 自动化学术研究
- 快速原型验证
- 小规模模型训练
- 科研自动化

### 📚 相关项目
- `aiming-lab/AutoResearchClaw` (⭐ 7,815) - 完全自主和自进化研究
- `uditgoenka/autoresearch` (⭐ 1,825) - Claude Code 自主目标导向迭代
