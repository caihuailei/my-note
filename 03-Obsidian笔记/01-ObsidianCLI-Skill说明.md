# Obsidian Official CLI Skill 使用指南

## 🎯 这是什么？

这是一个 OpenClaw 技能，让你的 AI 助手能够**完全控制你的 Obsidian 笔记系统**，通过命令行自动化所有笔记操作。

## ✨ 核心能力

你的 AI 现在可以帮你：

### 📝 自动化笔记创建
- 自动生成每日日记
- 创建会议记录
- 记录项目日志
- 添加待办事项

### ✅ 任务管理
- 列出所有待办任务
- 标记任务完成
- 追踪任务进度
- 按标签筛选任务

### 🔍 智能搜索
- 查找相关笔记
- 发现笔记之间的链接
- 找出孤立笔记
- 按内容搜索

### 🗄️ 数据库操作
- 查询 Obsidian 数据库
- 创建新条目
- 更新现有记录

### 🎨 工作区定制
- 安装插件
- 切换主题
- 保存工作区布局

## 🚀 如何使用

### 对 AI 说这些指令：

**创建日记：**
- "帮我创建今天的日记"
- "在日记中添加上午的任务"
- "记录今天的会议内容"

**管理任务：**
- "列出所有未完成的任务"
- "把日记中的第 3 个任务标记为完成"
- "查找所有带#工作标签的任务"

**创建笔记：**
- "创建一个关于项目 Alpha 的笔记"
- "创建会议记录，记录刚才的讨论"
- "帮我写一个学习笔记"

**搜索笔记：**
- "查找所有关于机器学习的笔记"
- "哪些笔记链接到了'项目概览'？"
- "找出所有孤立笔记"

**管理笔记：**
- "给这个笔记添加'重要'标签"
- "设置笔记类型为会议记录"
- "把这个笔记加入书签"

## 💡 使用技巧

### 让 AI 自动管理你的笔记

**每日工作流：**
```
"帮我做晨间例行：创建日记，添加今日目标，列出待办任务"
```

**会议记录：**
```
"创建一个会议笔记，包含：
- 参会人：张三、李四
- 讨论内容：项目进度汇报
- 下一步行动：下周完成原型"
```

**知识管理：**
```
"搜索所有关于人工智能的笔记，并找出它们之间的关联"
```

### 批量操作

AI 可以一次性执行多个命令：
- 创建日记 + 添加任务 + 设置标签
- 搜索笔记 + 创建链接 + 加入书签
- 安装插件 + 切换主题 + 保存工作区

## 🛠️ 故障排除

**问题：AI 说 Obsidian 未运行**
- 解决：打开 Obsidian 应用

**问题：找不到笔记**
- 解决：使用 `file=` 参数而不是完整路径
- 例如：`file="会议记录"` 而不是 `path="F:\biji\笔记\会议记录.md"`

**问题：命令执行失败**
- 确保 CLI 已启用
- 检查 Obsidian 版本是否为 1.12+
