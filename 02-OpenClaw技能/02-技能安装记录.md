- 从想法到论文，无需人工干预

### 🚀 使用方法
```
"Research quantum computing error correction"
"Research the impact of learning rate schedules on transformers"
"Research diffusion models for ICML 2026 submission"
```

### 🔄 23 阶段流程
1. TOPIC_INIT → 2. PROBLEM_DECOMPOSE
3. SEARCH_STRATEGY → 4. LITERATURE_COLLECT → 5. LITERATURE_SCREEN → 6. KNOWLEDGE_EXTRACT
7. SYNTHESIS → 8. HYPOTHESIS_GEN
9. EXPERIMENT_DESIGN → 10. CODE_GENERATION → 11. RESOURCE_PLANNING
12. EXPERIMENT_RUN → 13. ITERATIVE_REFINE
14. RESULT_ANALYSIS → 15. RESEARCH_DECISION
16. PAPER_OUTLINE → 17. PAPER_DRAFT → 18. PEER_REVIEW → 19. PAPER_REVISION
20. QUALITY_GATE → 21. KNOWLEDGE_ARCHIVE → 22. EXPORT_PUBLISH → 23. CITATION_VERIFY

### 📦 输出文件
- `paper_draft.md` - 完整学术论文
- `paper.tex` - 会议就绪的 LaTeX
- `references.bib` - 真实 BibTeX 引用
- `verification_report.json` - 引用验证报告
- `experiment runs/` - 实验代码和结果
- `charts/` - 自动生成的图表
- `reviews.md` - 同行评审
- `evolution/` - 自学习经验

4. **Plan Tests** - 创建 TEST.md，包含单元+E2E 测试计划
5. **Write Tests** - 实现全面的测试套件
6. **Document** - 更新 TEST.md 与测试结果
